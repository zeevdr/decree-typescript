# Quickstart

The simplest OpenDecree example — connect and read typed config values.

## What it shows

- `ConfigClient` with try/finally lifecycle
- Type converters: `Number`, `Boolean`, `String`
- `set()` for writing values

## Run

```bash
cd examples
npm install
npx tsx quickstart/main.ts
```
